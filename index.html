PROJECT

RETAIL SALES PERFORMANCE ANALYSIS

Overview

This project analyzes retail sales data to evaluate revenue trends and identify high-performing product categories.


Tools Used

  • SQL (SQLite)
  • Microsoft Excel

Key Tasks

  • Cleaned and analyzed sales data
  • Wrote SQL queries (GROUP BY, ORDER BY)
  • Created pivot tables and charts

Sample SQL Query

sql

SELECT category,
SUM(amount)
AS total_sales
FROM sales
GROUP BY category
ORDER BY total_sales
DESC;

Insights

  • Food category generated the highest revenue
  • Some days performed below average
  • Sales trend varied across categories

Column chart




Pivot table summary



Business Recommendation

The business can increase revenue by focusing on high-performing categories and introducing promotions on low-performing days.
